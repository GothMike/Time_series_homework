# Telegram-бот для анализа и прогнозирования акций  
**Учебный проект по анализу временных рядов**

## Описание проекта

Данный проект является **учебным** и предназначен для демонстрации навыков:
- работы с временными рядами;
- построения и сравнения моделей прогнозирования;
- разработки Telegram-ботов на Python;
- анализа результатов и формирования простых инвестиционных рекомендаций.

Проект **не является инвестиционной рекомендацией** и используется исключительно в учебных целях.

---

## Цель проекта

Разработать Telegram-бота, который:
1. Принимает от пользователя тикер компании и сумму условной инвестиции.
2. Загружает исторические котировки акций.
3. Обучает несколько моделей временных рядов разных классов.
4. Автоматически выбирает лучшую модель по метрикам качества.
5. Строит прогноз цены на 30 дней вперёд.
6. Формирует рекомендации по покупке и продаже.
7. Оценивает потенциальную прибыль при следовании стратегии.

---

## Функциональные возможности

- ввод тикера компании (например: `AAPL`, `MSFT`);
- ввод суммы условной инвестиции;
- обучение моделей:
  - классическая ML-модель (Ridge Regression с лагами);
  - статистическая модель (ARIMA);
  - нейросетевая модель (LSTM);
- сравнение моделей по метрикам (RMSE, MAPE);
- автоматический выбор лучшей модели;
- прогноз на 30 торговых дней;
- визуализация истории и прогноза;
- генерация инвестиционных рекомендаций;
- расчёт ориентировочной прибыли;
- логирование всех пользовательских запросов.
